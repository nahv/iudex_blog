Configuración local faltante. Para usar el admin desde tu máquina, copiá el template de env y completalo: 
cp public/js/env.example.js public/js/env.js
Después editá public/js/env.js y poné los valores reales de Supabase (Dashboard → Project Settings → API).

Solo emails autorizados pueden acceder. Si no tenés acceso pero deberías, contactá a un admin.
